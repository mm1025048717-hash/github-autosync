## 🚀 快速使用

### 图形界面版（最简单）

**方式1：根目录启动**
```powershell
# 在项目根目录
双击：启动图形界面.bat
```

**方式2：进入目录启动**
```powershell
cd 图形界面\app
双击：启动应用.bat
```

### 命令行版

```powershell
cd 脚本
.\start.ps1
```
