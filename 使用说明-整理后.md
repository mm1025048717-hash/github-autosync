## 🎮 使用流程

### 图形界面版

1. **启动应用**
   - 双击 `快速启动-图形界面.bat`

2. **配置设置**
   - 点击左侧"配置"菜单
   - 选择项目目录
   - 填写 GitHub Token
   - 点击"保存配置"

3. **启动同步**
   - 返回"首页"
   - 点击"启动自动同步"

4. **开始编码**
   - 在 Cursor 中编辑代码
   - 代码自动同步到 GitHub

### 命令行版

1. **进入脚本目录**
   ```powershell
   cd 脚本
   ```

2. **启动同步**
   ```powershell
   .\start.ps1
   ```

3. **停止同步**
   ```powershell
   .\stop.ps1
   ```
