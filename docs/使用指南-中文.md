# 📖 GitHub AutoSync 使用指南（中文版）

## 🎯 这是什么？

这是一个**自动同步工具**，让你可以：
- ✅ 一边在 Cursor 中聊天和写代码
- ✅ 一边自动将代码同步到 GitHub
- ✅ 不需要手动执行 git 命令

## 🚀 快速开始（3步）

### 第1步：安装

```powershell
# 运行安装脚本
.\install.ps1
```

### 第2步：配置 Token

```powershell
# 设置你的 GitHub Token
$env:GITHUB_TOKEN = "ghp_你的token"
```

### 第3步：启动

```powershell
# 启动自动同步
.\start.ps1
```

**完成！** 现在你可以开始编码了，代码会自动同步到 GitHub。

## 🎮 常用命令

```powershell
# 启动自动同步
.\start.ps1

# 停止自动同步
.\stop.ps1

# 或者按 Ctrl+C 停止

# 首次部署到 GitHub
.\deploy-with-token.ps1 -RepoUrl "https://github.com/用户名/仓库名.git" -Token "ghp_token"
```

## ❓ 常见问题

### Q: 文件变化后没有自动提交？

**A:** 可能原因：
1. 防抖时间还没到（默认10秒）
2. 文件在排除列表中
3. 服务没有运行

### Q: 如何获取 GitHub Token？

**A:** 
1. 访问 https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. 勾选 `repo` 权限
4. 生成并复制 Token

### Q: 如何停止服务？

**A:** 
- 前台运行：按 `Ctrl+C`
- 后台运行：运行 `.\stop.ps1`

## 📝 配置文件

创建 `config.json`（复制 `config.json.example`）：

```json
{
  "github": {
    "repository": "https://github.com/用户名/仓库名.git",
    "branch": "main",
    "token": "ghp_你的token"
  },
  "sync": {
    "debounceSeconds": 10
  }
}
```

## 🎯 使用场景

1. **与 AI 聊天写代码** - 在 Cursor 中聊天，代码自动同步
2. **多设备协作** - 一台设备编辑，其他设备拉取
3. **自动备份** - 每次保存自动提交，相当于自动备份

## ⚠️ 注意事项

- Token 不要提交到代码仓库
- 建议使用环境变量存储 Token
- 确保 `.gitignore` 已正确配置
