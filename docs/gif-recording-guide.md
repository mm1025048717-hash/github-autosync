# GIF 录制指南

## 🎬 推荐录制流程（15-30 秒）

### 场景 1：完整工作流演示（推荐）

**步骤：**
1. **打开应用** (2秒)
   - 显示 Apple 风格的界面
   - 展示"检测中"状态

2. **配置项目** (3秒)
   - 点击"浏览"选择项目文件夹
   - 输入 GitHub Token（可以模糊处理）
   - 点击"开始同步"

3. **修改代码** (5秒)
   - 打开一个代码文件（如 `main.js`）
   - 添加几行代码（例如：添加一个新函数）
   - 保存文件

4. **AI 生成 Commit** (5秒)
   - 显示活动日志："检测到文件变化..."
   - 显示："AI 分析中..."
   - 显示生成的 commit message（例如：`feat: add user authentication function`）
   - 显示："已提交并推送"

5. **查看历史记录** (3秒)
   - 滚动到历史记录时间轴
   - 展示新生成的 commit
   - 展示"恢复"按钮

6. **演示回滚** (2秒)
   - 点击"恢复"按钮
   - 显示确认对话框
   - 显示"已恢复到指定版本"

**总时长：** 约 20 秒

## 🛠️ 录制工具推荐

### Windows
- **ScreenToGif** (免费，推荐)
  - 下载：https://www.screentogif.com/
  - 优点：轻量、易用、可编辑
  - 输出：高质量 GIF

- **ShareX** (免费)
  - 下载：https://getsharex.com/
  - 优点：功能强大、支持快捷键
  - 输出：GIF/视频

- **OBS Studio** (免费)
  - 下载：https://obsproject.com/
  - 优点：专业级录制
  - 输出：视频（需转换）

### macOS
- **Kap** (免费，推荐)
  - 下载：https://getkap.co/
  - 优点：专为 macOS 设计、简单易用
  - 输出：GIF/视频

- **CleanShot X** (付费，但功能强大)
  - 下载：https://cleanshot.com/
  - 优点：专业、功能丰富
  - 输出：高质量 GIF

### 在线工具
- **CloudApp** - 录制并自动生成 GIF
- **LICEcap** - 简单易用的 GIF 录制工具

## 📐 录制设置

### 分辨率
- **推荐：** 1920x1080 (Full HD)
- **最低：** 1280x720 (HD)
- **注意：** 确保文字清晰可读

### 帧率
- **推荐：** 10-15 FPS（GIF 文件较小）
- **流畅：** 20-30 FPS（文件较大但更流畅）

### 录制区域
- **全屏：** 展示完整应用界面
- **窗口：** 只录制应用窗口（推荐，文件更小）

## ✨ 录制技巧

### 1. 准备演示环境
- [ ] 关闭不必要的应用
- [ ] 清理桌面（隐藏敏感信息）
- [ ] 准备一个测试项目
- [ ] 确保网络连接正常（用于推送）

### 2. 突出关键点
- **AI 生成过程** - 这是核心卖点
- **时间轴界面** - 展示可视化历史
- **回滚功能** - 展示备份工具的核心价值

### 3. 速度控制
- **正常速度：** 关键步骤（AI 生成、回滚）
- **加速：** 等待过程（文件监听、网络请求）
- **暂停：** 重要信息（commit message、时间轴）

### 4. 添加标注（可选）
- 使用箭头指向关键按钮
- 高亮显示 AI 生成的 commit message
- 标注"自动"、"AI 生成"等关键词
