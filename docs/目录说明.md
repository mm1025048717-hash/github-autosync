## 📋 文件功能说明

### 🎯 核心文件（必须了解）

| 文件名 | 中文说明 | 用途 | 重要程度 |
|--------|---------|------|---------|
| `auto-sync.ps1` | **自动同步主程序** | 监听文件变化，自动提交推送 | ⭐⭐⭐⭐⭐ |
| `start.ps1` | **启动脚本** | 一键启动自动同步服务 | ⭐⭐⭐⭐ |
| `stop.ps1` | **停止脚本** | 停止自动同步服务 | ⭐⭐⭐ |
| `deploy-with-token.ps1` | **部署脚本** | 首次部署到 GitHub | ⭐⭐⭐⭐ |
| `install.ps1` | **安装脚本** | 自动安装和配置 | ⭐⭐⭐ |

## 🚀 使用流程

### 第一次使用

1. **阅读文档** → `README.md` 或 `docs/QUICKSTART.md`
2. **运行安装** → `install.ps1`
3. **配置仓库** → `deploy-with-token.ps1`
4. **启动服务** → `start.ps1`

### 日常使用

1. **启动** → `start.ps1`
2. **停止** → `stop.ps1` 或按 `Ctrl+C`
