## 🚀 发布步骤

### 步骤1: 在 GitHub 创建仓库

1. 访问 https://github.com/new
2. 填写信息：
   - Repository name: `github-autosync`
   - Description: `🔄 聊天式自动同步工具 - 一边聊天，一边自动同步到 GitHub（支持图形界面）`
   - 选择 Public
   - **不要**勾选 "Initialize this repository with a README"
3. 点击 "Create repository"

### 步骤2: 初始化并推送

```powershell
cd C:\Users\10250\OneDrive\Desktop\GitHub-AutoSync

# 初始化 Git（如果还没有）
git init
git branch -M main

# 添加所有文件
git add .

# 提交
git commit -m "🎉 初始版本发布 - 包含命令行版和图形界面版"

# 添加远程仓库
git remote add origin https://github.com/mm1025048717-hash/github-autosync.git

# 推送（使用 Token）
.\deploy-with-token.ps1 -RepoUrl "https://github.com/mm1025048717-hash/github-autosync.git" -Token "YOUR_TOKEN_HERE"
```

### 步骤3: 完善仓库信息
