# 📋 GitHub AutoSync 项目说明

## 🎯 项目目标

创建一个简单易用的 GitHub 自动同步工具，让开发者可以：
- ✅ 一边在 Cursor 等编辑器中聊天和编码
- ✅ 一边自动将代码同步到 GitHub
- ✅ 无需手动执行 git 命令
- ✅ 零配置，开箱即用

## 📝 使用说明

### 用户如何使用

1. **克隆仓库**
   ```bash
   git clone https://github.com/mm1025048717-hash/github-autosync.git
   cd github-autosync
   ```

2. **运行安装脚本**
   ```powershell
   .\install.ps1
   ```

3. **配置 Token**
   ```powershell
   $env:GITHUB_TOKEN = "ghp_your_token"
   ```

4. **启动自动同步**
   ```powershell
   .\start.ps1
   ```

## ✨ 核心功能

1. **文件监听** - 使用 PowerShell FileSystemWatcher
2. **防抖机制** - 避免频繁提交
3. **自动提交** - git add → commit → push
4. **智能排除** - 自动排除不需要的文件
5. **安全认证** - 使用 GitHub Token
