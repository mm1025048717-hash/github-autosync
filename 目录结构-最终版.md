## 🚀 快速使用

### 方式一：图形界面版（推荐新手）⭐

**最简单：**
```
双击：快速启动-图形界面.bat
```

**或进入目录：**
```powershell
cd app
双击：启动应用.bat
```

### 方式二：命令行版（推荐高级用户）

```powershell
cd 脚本
.\start.ps1
```
